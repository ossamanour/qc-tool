## Requirement

- yarn 1.22.22
- node 22.14.0
- npm 10.9.2
- nvm 0.40.2
  This project is built using Vite and developed with yarn.

## Usage Instruction

- Navigate to the `frontend` folder, use the install command to install the app.

```
yarn install
```

- Run the following command to start the interface.

```
yarn run dev
```
